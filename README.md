# Prompty z filmów

Zestaw promptów do odtworzenia projektów omawianych w filmach instruktażowych.

Każdy folder odpowiada narzędziu. W środku znajdziesz pliki `.md` z gotowymi promptami do skopiowania.

## Zawartość

- **[Claude-Code/](Claude-Code/)** — projekty budowane w Claude Code
- **[Python/](Python/)** — skrypty i instrukcje Pythona
- **[Lovable/](Lovable/)** — projekty budowane w Lovable
- **[NotebookLM/](NotebookLM/)** — przewodniki do NotebookLM

## Jak używać

Otwórz plik odpowiadający filmowi. Każdy krok ma blok kodu z przyciskiem kopiowania — kliknij go i wklej prompt w narzędziu.
